class modules.jedli_logic.Highlighter(variables)¶
attribute_colors(AND_OR_dics)¶

Attribute colors and hidden signs to every set of search words. NOT words don’t get colors and hidden signs; OR words get the same colors/hidden signs as the previous AND set Returns: * regexesPerColor: a dictionary that contains the following pairs

  • key: color (an html color code)
  • value: a list containing the following elements:
    1. a list of the regular expressions that should be highlighted in this color
    2. the name of the checklist (if any; otherwise: None)
    3. a list of the original words that form the basis of the regular expressions
    4. the hidden sign that will be attributed to this batch of words
  • regexes: a dictionary that contains the following pairs:
    • key: regex (a regular expression that should be highlighted)
    • value: a list containing the following elements:
      1. the color this regex should be highlighted in
      2. the name of the checklist the regex belongs to (if any; otherwise: None)
      3. the hidden sign that will be attributed to this regex
